**新增支付方式选择**:
```vue

  <el-radio label="wechat">微信支付</el-radio>
  <el-radio label="alipay">支付宝支付</el-radio>
  <el-radio label="bank">银行卡支付</el-radio>
</el-radio-group>
```

**智能支付流程**:
1. **余额支付**: 直接扣款，实时到账
2. **第三方支付**: 二次确认 → 跳转支付平台

**关键代码**:
```javascript
// 非余额支付时提示用户
if (this.payMethod !== 'balance') {
  const payMethodNames = {
    'wechat': '微信支付',
    'alipay': '支付宝支付',
    'bank': '银行卡支付'
  }
  this.$confirm(`即将跳转至${payMethodNames[this.payMethod]}，请确认订单信息`, 
    '支付提示', {
      confirmButtonText: '确定',
      cancelButtonText: '取消',
      type: 'warning'
  }).then(() => {
    this.processPayment()
  }).catch(() => {})
} else {
  this.processPayment()
}
```


**特点**:

```
选择商品 → 填写收货地址 → 选择第三方支付 → 二次确认 →

```

**特点**:
- ✅ 多种支付选择
- ✅ 安全确认
- ✅ 模拟跳转提示
- ✅ 易于对接真实支付接口

### 数据结构

```javascript

  return {
    payMethod: 'balance', // 支付方式：balance | wechat | alipay | bank
    // ... 其他数据
  }

```

### 核心方法

#### 1. `confirmPay()` - 确认支付
- 验证收货地址
- 检查余额是否充足
- 判断是否需要二次确认

#### 2. `processPayment()` - 处理支付
- 提交支付请求到后端
- 处理支付结果
- 模拟跳转第三方支付

## 📊 支付方式对比

| 支付方式 | 优点 | 缺点 | 适用场景 |
|---------|------|------|---------|
| 余额支付 | 实时到账、无需跳转 | 需要先充值 | 小额支付、快速结算 |
| 微信支付 | 用户量大、便捷 | 需要跳转 | 日常购物 |
| 支付宝 | 安全可靠、普及率高 | 需要跳转 | 大额支付 |
| 银行卡 | 额度高、稳定 | 手续相对复杂 | 大额消费 |

```java
public void pay(@RequestBody Orders orders) {
```
